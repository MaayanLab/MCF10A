File Format

GR

GR metrics are derived from normalized growth rate inhibition (GR) values which are based on the ratio of growth rates in the presence and absence of perturbagen. Largely independent of cell division rate and assay duration, GR metrics are more robust than IC50 and Emax for assessing cellular response to drugs, RNAi, and other perturbations in which control cells divide over the course of the assay.

Cell Counts

Columns of the file include:

  • cell_line
  • agent
  • concentration
  • GRvalue
  • cell_count
  • cell_count__ctrl
  • cell_count__time0
  • Replicate

GR Metrics

Columns of the file include:

  • cell_line
  • agent
  • time
  • GR50
  • GRmax
  • GR_AOC
  • GEC50
  • GRinf
  • Hill
  • Replicate

L1000

L1000 is a gene expression profiling assay based on the direct measurement of a reduced representation of the transcriptome and computational inference of the portion of the transcriptome not explicitly measured.

GCT

GCT is a matrix tab delimited file format. In addition to a matrix of numerical data a gct file contains row and column ids and annotations. Learn more

P100

P100 assay consists of a set of 96 phosphopeptide probes that are easily observable and quantifiable in diverse cell types ranging from embryonic stem cells to human leukemia cells, and that they behave in interesting and orthogonal ways in response to varied chemical and genetic pertrubations.

GCT

GCT is a matrix tab delimited file format. In addition to a matrix of numerical data a gct file contains row and column ids and annotations. Learn more

Cyclic Immunofluorescence (CycIF)

Cyclic immunofluorescence (CycIF) is a fluorescence imaging assay that measures protein levels and phosphorylation states in cells.

CycIF Mean ZScores

File contains matrix of mean zscores of immunofluorescent intensity measurements for each combination of agent, concentration, time, and compartment (cytosol or nucleus). Zscoring implemented for each combination using the mean and standard deviation of that combination's compartment.

Columns of the file include:

  • agent
  • concentration
  • time
  • compartment

CycIF Raw

Data file contains matrix of immunofluorescent intensity measurements for cells treated with different doses of a perturbagen. See datacolumn_LUT Excel file in downloaded dataset results directory from HMS LINCS for additional column header details.

Microwestern Arrays (MWA)

Microwestern Arrays (MWA) is a high-throughput variation of the Western blot, which uses electrophoresis to determine protein levels and modifications by measuring fluorescence intensity.

MWA Measurements

File contains matrices of fluorescence intensity measurements of a target protein and alpha tubulin for cells treated with different doses of a perturbagen, as well as the ratios between target protein and alpha tubulin for each perturbation.

Columns of the file include:

  • Ave
  • S.D.
  • C.V.
  • ss

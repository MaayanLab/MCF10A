File Format

GR

GR metrics are derived from normalized growth rate inhibition (GR) values which are based on the ratio of growth rates in the presence and absence of perturbagen. Largely independent of cell division rate and assay duration, GR metrics are more robust than IC50 and Emax for assessing cellular response to drugs, RNAi, and other perturbations in which control cells divide over the course of the assay.

Cell Counts

Columns of the file include:

  • cell_line
  • agent
  • concentration
  • GRvalue
  • cell_count
  • cell_count__ctrl
  • cell_count__time0
  • Replicate

GR Metrics

Columns of the file include:

  • cell_line
  • agent
  • time
  • GR50
  • GRmax
  • GR_AOC
  • GEC50
  • GRinf
  • Hill
  • Replicate

L1000

L1000 is a gene expression profiling assay based on the direct measurement of a reduced representation of the transcriptome and computational inference of the portion of the transcriptome not explicitly measured.

GCT

GCT is a matrix tab delimited file format. In addition to a matrix of numerical data a gct file contains row and column ids and annotations. Learn more

P100

P100 assay consists of a set of 96 phosphopeptide probes that are easily observable and quantifiable in diverse cell types ranging from embryonic stem cells to human leukemia cells, and that they behave in interesting and orthogonal ways in response to varied chemical and genetic pertrubations.

GCT

GCT is a matrix tab delimited file format. In addition to a matrix of numerical data a gct file contains row and column ids and annotations. Learn more

Row headers include:

  • pr_gene_symbol

  • pr_p100_original_probe_id
  • pr_p100_phosphosite

  • pr_probe_suitability_manual
  • pr_uniprot_id

Global chromatin profiling (GCP) technology is a mass spectrometry-based assay to identify and quantify post-translational modifications on histone proteins from bulk chromatin.
